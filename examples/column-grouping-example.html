DataGrid5 - 列ヘッダグルーピング

Column Grouping Example - Multi-level hierarchical headers

📊 列グルーピングについて / About Column Grouping:
  • 2レベルヘッダ: グループヘッダ（1行目） + 個別カラム（2行目）
  • 3レベル以上: 階層的なグループ構造をサポート
  • API使用例: grid.add_column_group("Tokyo", 0, 3, 0)
  • level=0: 最上位グループ、level=1: 第2レベルグループ
コード例:
grid.add_column_group("Tokyo", 0, 3, 0); // 列0-3をTokyoグループに
grid.add_column_group("Osaka", 4, 9, 0); // 列4-9をOsakaグループに
grid.set_header_row_height(35); // ヘッダ行の高さ調整
ヘッダレベル: 1
ヘッダ高さ: 30px
グループ数: 0
