🔒 読み取り専用列設定

Read-only Column Configuration Example

特定の列を読み取り専用に設定できます。読み取り専用列はダブルクリックしても編集できません。 このサンプルでは、ID列・作成日・更新日を読み取り専用に設定しています。

📊 列の編集可否状態

💡 使い方

  • 赤色の列（読み取り専用）はダブルクリックしても編集できません
  • 緑色の列（編集可能）はダブルクリックで編集できます
  • ボタンをクリックして列の編集可否を動的に変更できます
  • 読み取り専用列を編集しようとすると、メッセージが表示されます
